About

Web Color Checker

A tool for checking accessibility properties of color schemes for the web.

Contrast ratios are calculated using the WCAG 2 algorithm assuming an sRGB color space. Target values for good readability are 4.5 or better, with 7 or better being ideal.

The various samples of color-blind appearances for colors are computed using the HCIRN color blindness simulation function, which is known to not be perfectly accurate but provides a reasonable approximation for designer purposes of what a person with that type of color blindness would see.
